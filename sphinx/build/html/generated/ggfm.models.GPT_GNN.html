Parameters:

  • gnn (class:ggfm.models) – The used GNN model.

  • rem_edge_list (dict) – The remaining edge list after sampling.

  • attr_decoder (ggfm.models) – Attribute decoder.

  • neg_samp_num (int) – Maximum number of negative sample for each target node. (default: 1)

  • device (int) – Device

  • neg_queue_size (int, optional) – Max size of negetive adaptive embedding queue. (default: 0)
